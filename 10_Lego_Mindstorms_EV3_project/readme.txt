Installation der EV3 Roboter Applikation (see English Version below)
===================================================================
1) Lego Mindstorms Entwicklungsumgebung starten
2) Datei PathRunner_V1.ev3 öffnen
3) Projekt auf den Roboter laden
4) Programm "Main" starten

Hinweis: Siehe cargo_area1.jpg und cargo_area2.jpg für den 
         Bau der Ladeplattform.

========================English Version====================================
Installation of the EV3 robot application
--------------------------------------------
1) Start Lego Mindstorms development environment
2) Open file PathRunner_V1.ev3 
3) Load project on the robot
4) Start "Main" 

Hint: See cargo_area1.jpg and cargo_area2.jpg for 
      the cargo area.
